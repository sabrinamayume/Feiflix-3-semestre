PLATAFORMA DE FILMES DO STUDIO GHIBLI	

Aluna: Sabrina Mayume Silva Uezu

Curso: Ciências da Computação – 3° Semestre


Tecnologias utilizadas: 

	•	Linguagem Java

	•	JDBC PostgreSQL

	•	MVC (Model, View e Controller)

Funcionalidades:

	•	Cadastrar novo usuário

	•	Login de usuário

	•	Buscar vídeo por nome

	•	Listar informações de vídeos buscados

	•	Curtir e descurtir vídeos 

	•	Gerenciar favoritos 

		o	Criar, editar, excluir lista de reprodução de vídeos favoritos

		o	Adicionar /remover vídeos da lista de favoritos 

Resumo: 
O projeto foi desenvolvido individualmente, tendo como objetivo apresentar uma plataforma semelhante a Crunchyroll, voltada somente aos animes do Studio Ghibli.
